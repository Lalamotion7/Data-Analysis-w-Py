# 📊 Portfólio de Análise de Dados

Bem-vindo ao meu repositório central de projetos de dados! 

Aqui você encontrará desenvolvimentos focados em extração, tratamento e visualização de dados. O objetivo deste portfólio é demonstrar a aplicação prática de tecnologias analíticas para resolver problemas reais de negócio.

Minha vivência técnica em infraestrutura e suporte corporativo me proporcionou uma base sólida em resolução de problemas complexos, pensamento lógico e visão sistêmica — habilidades que agora aplico diretamente na construção de pipelines de dados e dashboards estratégicos.

## 🛠️ Tecnologias e Ferramentas
* **Linguagens:** Python (Pandas, NumPy, SQLAlchemy), SQL (PostgreSQL, CTEs, Window Functions).
* **Visualização de Dados:** Power BI (DAX, Modelagem, UI/UX, Data Storytelling).
* **Processos:** ETL, Limpeza de Dados, Análise Exploratória.
